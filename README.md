# Введение 
**Калькулятор индекса массы тела - (ИМТ)** - это интрумент, предназначенный для быстрой оценки соответствия массы человека его росту. Продукт позволяет пользователям определить,  является ли их вес недостаточным, нормальным, или избыточным, что помогает в базовом мониторинге состояния здоровья.

Расчёт производится по классической формуле: 
ИМТ = масса тела (кг) / рост² (м²)
