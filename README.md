# Введение 
**Калькулятор индекса массы тела - (ИМТ)** - это интрумент, предназначенный для быстрой оценки соответствия массы человека его росту. Продукт позволяет пользователям определить,  является ли их вес недостаточным, нормальным, или избыточным, что помогает в базовом мониторинге состояния здоровья.

Расчёт производится по классической формуле: 
ИМТ = масса тела (кг) / рост² (м²)

---

# Примеры пользования 
1. **Ввод данных:** При вводе значения в поле "Рост" данные сохраняются в памяти приложения
2. **Активация:** При вводе значения в поле "Вес" активируется кнопка **"Рассчитать"**
3. **Вычисление:** При нажатии на кнопку **"Рассчитать"** приложение вычисляет индекс и выводит результат на экран.
4. **Сброс:** При нажатии кнопки **"Очистить"** все введённые данные в полях сбрасываются до нуля
5. **Индикация:** При получении результата цветом выделяется соответствующая категория (например *зеленый* для нормы).
6. **Валидация:** При попытке ввести текст вмместо цифр поле ввода подсвечивается **красным цветом**.
7. **История:** При нажатии на кнопку **"Сохранить результат"** данные записываются в локальную историю браузера.
